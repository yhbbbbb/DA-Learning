# Tableau 看板搭建指南

## 1. 数据连接

### 连接方式
1. 打开 Tableau Desktop
2. 选择 **"文本文件"** → 连接 `data/user_behavior_clean.csv`
3. 或选择 **"Microsoft SQL Server"** → 连接 SQLite 数据库（需安装驱动）

### 字段配置
| 字段名 | 数据类型 | 说明 |
|--------|----------|------|
| user_id | 字符串 | 用户ID |
| item_id | 字符串 | 商品ID |
| category_id | 字符串 | 商品类目ID |
| behavior_type | 字符串 | 行为类型 |
| datetime | 日期时间 | 行为时间 |
| date | 日期 | 行为日期 |
| hour | 整数 | 小时 |
| weekday | 整数 | 星期几 |

### 计算字段（在Tableau中创建）
```
# 行为类型中文名
行为名称 = CASE [behavior_type]
    WHEN "pv" THEN "浏览"
    WHEN "cart" THEN "加购"
    WHEN "fav" THEN "收藏"
    WHEN "buy" THEN "购买"
END

# 转化率
转化率 = COUNT(IF [behavior_type]="buy" THEN [user_id] END) /
         COUNT(IF [behavior_type]="pv" THEN [user_id] END) * 100
```

## 3. 交互筛选器设置

### 添加筛选器
1. **行为类型筛选器**：允许用户选择查看特定行为
2. **日期范围筛选器**：选择分析的时间范围
3. **品类筛选器**：筛选特定商品类目
4. **用户分群筛选器**：查看特定用户群体

### 筛选器联动
- 将筛选器设置为 **"应用于工作表"** → **"使用此数据源的所有项"**
- 这样一个筛选器的变化会影响所有相关图表

## 4. 样式美化建议

### 配色方案
- 浏览：#3498db（蓝色）
- 加购：#2ecc71（绿色）
- 收藏：#f39c12（橙色）
- 购买：#e74c3c（红色）

### 图表类型建议
| 分析内容 | 推荐图表类型 |
|---------|-------------|
| 转化漏斗 | 漏斗图 / 横向条形图 |
| 趋势分析 | 折线图 |
| 时段分析 | 柱状图 |
| 用户分群 | 饼图 / 树状图 |
| RFM散点 | 散点图 |
| 品类对比 | 条形图 |

## 5. 导出与分享

### 导出方式
1. **Tableau Public**：免费发布到 Tableau Public 平台（简历展示用）
2. **PDF导出**：导出为PDF用于报告
3. **图片导出**：导出为PNG用于PPT

### 简历展示建议
- 将 Tableau Public 链接放在简历中
- 截取关键图表放入简历的项目经历部分
- 准备好在面试中讲解每个图表的业务含义
